# ملخص الدرس

## لماذا Python في الذكاء الاصطناعي؟

- سهلة القراءة والبدء.
- تستخدم نفس اللغة من الأساسيات إلى تحليل البيانات وتعلم الآلة.
- توفر مكتبات قوية مثل Pandas وNumPy وscikit-learn.
- مناسبة للتجربة السريعة والمشاريع التعليمية.

## قبل الدخول إلى Machine Learning

تأكد أنك تستطيع:

- استخدام المتغيرات والقوائم والدوال.
- قراءة ملف CSV.
- فهم الصفوف والأعمدة.
- تثبيت مكتبة عبر pip.
- قراءة بيانات بسيطة باستخدام Pandas.

## الخطوة التالية

انتقل إلى تحليل البيانات ببايثون، ثم ابدأ بتعلم التدريب والاختبار والتنبؤ في Machine Learning.
