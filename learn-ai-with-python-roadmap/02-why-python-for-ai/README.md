# لماذا تستخدم Python في الذكاء الاصطناعي؟

ملفات تطبيقية للدرس الثاني من مسار **الذكاء الاصطناعي ببايثون** في موقع **بايثون العرب**.

## رابط الدرس
https://www.arabpython.com/p/why-python-for-artificial-intelligence.html

## محتويات الحزمة

- `lesson_code.py`: مثال مبسط يوضح قراءة بيانات الطلاب وحساب متوسط الدرجات.
- `practice.py`: تمرين عملي للمستخدم.
- `solution.py`: حل التمرين.
- `data/students.csv`: بيانات تدريبية صغيرة.
- `notes.md`: ملخص الدرس وخطوات التعلم.
- `lesson-guide.pdf`: كتيب شرح مختصر.
- `requirements.txt`: المكتبات المطلوبة.

## التشغيل

ثبّت المكتبة المطلوبة:

```bash
pip install -r requirements.txt
```

ثم شغّل ملف الدرس:

```bash
python lesson_code.py
```

## الفكرة التعليمية

Python مناسبة للمبتدئ في الذكاء الاصطناعي لأنها واضحة القراءة وتوفر مكتبات قوية لتحليل البيانات وتعلم الآلة. في هذه الحزمة ستبدأ بقراءة بيانات بسيطة، حساب متوسطات، ثم استخدام Pandas لفهم الجدول.

## ملاحظة

هذا المثال تعليمي فقط، ولا يمثل نموذجًا حقيقيًا لاتخاذ قرارات حول الطلاب أو الدرجات.
