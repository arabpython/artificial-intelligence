# أفضل مكتبات Python للذكاء الاصطناعي للمبتدئين

حزمة تطبيقية مرافقة للدرس الرابع من مسار **الذكاء الاصطناعي ببايثون** في **بايثون العرب**.

## رابط الدرس

https://www.arabpython.com/2026/06/best-python-libraries-for-artificial-intelligence-beginners.html

## محتويات الحزمة

- `lesson_code.py`: مثال يجمع Pandas وNumPy وMatplotlib.
- `practice.py`: تمرين للمبتدئ.
- `solution.py`: حل التمرين.
- `data/students.csv`: بيانات تدريبية صغيرة.
- `notes.md`: ملخص دور كل مكتبة وترتيب تعلمها.
- `lesson-guide.pdf`: كتيب شرح PDF.
- `requirements.txt`: المكتبات المطلوبة.

## التثبيت والتشغيل

```bash
pip install -r requirements.txt
python lesson_code.py
```

سيظهر رسم بياني باسم `students_scores_chart.png`.

## ترتيب التعلم المقترح

1. NumPy
2. Pandas
3. Matplotlib
4. scikit-learn
5. TensorFlow أو PyTorch لاحقًا
