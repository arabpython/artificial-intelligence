# ماذا أتعلم قبل دخول مجال الذكاء الاصطناعي؟

ملفات تطبيقية للدرس الثالث من مسار **الذكاء الاصطناعي ببايثون** في موقع **بايثون العرب**.

## رابط الدرس

https://www.arabpython.com/2026/06/what-to-learn-before-artificial-intelligence.html

## محتويات الحزمة

- `lesson_code.py`: مثال تدريجي يقرأ ملف درجات ويحسب مؤشرات بسيطة.
- `practice.py`: تمرين عملي للمستخدم.
- `solution.py`: حل التمرين.
- `data/students.csv`: بيانات تدريبية صغيرة.
- `notes.md`: ملخص الدرس وخريطة التعلم.
- `lesson-guide.pdf`: كتيب شرح PDF مختصر.
- `requirements.txt`: المكتبات المطلوبة.

## التشغيل

ثبّت المكتبة المطلوبة:

```bash
pip install -r requirements.txt
```

ثم شغّل مثال الدرس:

```bash
python lesson_code.py
```

## الفكرة التعليمية

قبل الدخول إلى الذكاء الاصطناعي، ركّز على:
1. أساسيات Python.
2. قراءة الملفات والجداول.
3. فهم المتوسطات والقيم الأساسية.
4. تحليل بيانات صغيرة.
5. الانتقال تدريجيًا إلى تعلم الآلة.

## ملاحظة

هذه الحزمة تعليمية. لا تستخدم بيانات المثال لاتخاذ قرارات حقيقية.
