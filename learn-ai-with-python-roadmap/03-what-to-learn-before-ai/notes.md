# ملخص الدرس الثالث

## ماذا أتعلم قبل دخول مجال الذكاء الاصطناعي؟

لا تحتاج إلى معرفة كل شيء مسبقًا. ابدأ بهذا التسلسل:

1. **أساسيات Python**
   - المتغيرات
   - الشروط
   - الحلقات
   - الدوال
   - القوائم والقواميس

2. **الملفات والبيانات**
   - ملفات CSV
   - الصفوف والأعمدة
   - القيم الناقصة
   - قراءة البيانات باستخدام Pandas

3. **الرياضيات العملية**
   - المتوسط
   - النسب المئوية
   - الإحصاء الأساسي
   - الاحتمالات والجبر تدريجيًا

4. **Machine Learning Basics**
   - التدريب
   - الاختبار
   - الميزات Features
   - الهدف Target
   - التوقع والتصنيف

## قاعدة مهمة

ابدأ بمشروع صغير واضح. فهم البيانات أهم من نسخ نموذج كبير لا تعرف كيف يعمل.

## الخطوة التالية

بعد إنهاء هذه الحزمة، تابع إلى:
- تحليل البيانات ببايثون.
- تعلم الآلة ببايثون للمبتدئين.
- مشروع AI بسيط خطوة بخطوة.
