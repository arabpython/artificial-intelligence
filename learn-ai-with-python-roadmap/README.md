<h2>مراحل المسار</h2>

<h3>1. مراجعة أساسيات Python</h3>

<p>
قبل الدخول إلى الذكاء الاصطناعي، يجب أن يكون الطالب مرتاحًا مع أهم أساسيات Python، مثل:
</p>

<ul>
  <li>المتغيرات.</li>
  <li>أنواع البيانات.</li>
  <li>الشروط.</li>
  <li>الحلقات.</li>
  <li>الدوال.</li>
  <li>القوائم.</li>
  <li>القواميس.</li>
  <li>التعامل مع الملفات.</li>
</ul>

<h3>2. التعامل مع البيانات</h3>

<p>
الذكاء الاصطناعي يعتمد على البيانات، لذلك يتعلم الطالب هنا كيف يقرأ البيانات ويتعامل معها داخل Python.
</p>

<ul>
  <li>قراءة الملفات النصية.</li>
  <li>قراءة ملفات CSV.</li>
  <li>قراءة ملفات JSON.</li>
  <li>تنظيف البيانات البسيطة.</li>
  <li>تجهيز البيانات للتطبيقات العملية.</li>
</ul>

<h3>3. أساسيات تحليل البيانات</h3>

<p>
قبل تدريب أي نموذج، يجب فهم البيانات أولًا.
</p>

<ul>
  <li>حساب المتوسطات.</li>
  <li>استخراج أكبر وأصغر قيمة.</li>
  <li>عدّ العناصر وتصنيفها.</li>
  <li>تلخيص البيانات.</li>
  <li>قراءة النتائج بطريقة مفهومة.</li>
</ul>

<h3>4. أساسيات NumPy</h3>

<p>
مكتبة NumPy من المكتبات الأساسية في مجال البيانات والذكاء الاصطناعي.
</p>

<ul>
  <li>إنشاء المصفوفات Arrays.</li>
  <li>العمليات الحسابية على المصفوفات.</li>
  <li>التعامل مع الأبعاد.</li>
  <li>استخدام NumPy في الأمثلة العملية.</li>
</ul>

<h3>5. أساسيات Pandas</h3>

<p>
مكتبة Pandas تساعد الطالب على التعامل مع الجداول والبيانات المنظمة.
</p>

<ul>
  <li>قراءة ملفات CSV.</li>
  <li>إنشاء DataFrame.</li>
  <li>عرض البيانات.</li>
  <li>فلترة الصفوف.</li>
  <li>اختيار الأعمدة.</li>
  <li>تنظيف البيانات البسيطة.</li>
</ul>

<h3>6. عرض البيانات Data Visualization</h3>

<p>
عرض البيانات يساعد على فهم الأنماط والنتائج بصريًا قبل بناء النماذج.
</p>

<ul>
  <li>رسم مخططات بسيطة.</li>
  <li>فهم العلاقة بين القيم.</li>
  <li>استخدام الرسوم لتوضيح النتائج.</li>
</ul>

<h3>7. أساسيات تعلم الآلة Machine Learning</h3>

<p>
في هذه المرحلة يبدأ الطالب بفهم أساسيات تعلم الآلة وبناء نماذج بسيطة باستخدام Python.
</p>

<ul>
  <li>ما هو تعلم الآلة؟</li>
  <li>الفرق بين البيانات التدريبية والاختبارية.</li>
  <li>تجهيز البيانات للنموذج.</li>
  <li>تدريب نموذج بسيط.</li>
  <li>اختبار النموذج.</li>
  <li>فهم دقة النموذج ونتائجه.</li>
</ul>

<h3>8. مشاريع تطبيقية بسيطة</h3>

<p>
بعد فهم الأساسيات، ينتقل الطالب إلى مشاريع مصغرة تجمع بين Python والبيانات وتعلم الآلة.
</p>

<ul>
  <li>توقع نتيجة طالب.</li>
  <li>تحليل بيانات مبيعات.</li>
  <li>تصنيف نص بسيط.</li>
  <li>توقع قيمة بناءً على بيانات.</li>
  <li>مشروع صغير باستخدام Pandas.</li>
  <li>مشروع بسيط باستخدام نموذج Machine Learning.</li>
</ul>

<h2>طريقة تشغيل الملفات</h2>

<p>
لتشغيل أي ملف Python داخل هذا المسار:
</p>

<pre dir="ltr" align="left"><code>python file_name.py</code></pre>

<p>
مثال:
</p>

<pre dir="ltr" align="left"><code>python examples/example_01.py</code></pre>

<p>
إذا كان الدرس يحتاج مكتبات خارجية، ستجد ملف:
</p>

<pre dir="ltr" align="left"><code>requirements.txt</code></pre>

<p>
ويمكن تثبيت المتطلبات باستخدام:
</p>

<pre dir="ltr" align="left"><code>pip install -r requirements.txt</code></pre>
