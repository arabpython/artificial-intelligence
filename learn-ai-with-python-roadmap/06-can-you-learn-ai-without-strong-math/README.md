# هل يمكن تعلم الذكاء الاصطناعي بدون رياضيات قوية؟

حزمة تطبيقية مرافقة للدرس السادس من مسار **الذكاء الاصطناعي ببايثون** في **بايثون العرب**.

## رابط الدرس

https://www.arabpython.com/2026/06/can-you-learn-artificial-intelligence-without-strong-math.html

## محتويات الحزمة

- `lesson_code.py`: أمثلة عملية على المتوسط والانحراف المعياري والاحتمال والمصفوفات.
- `practice.py`: تمرين تطبيقي للمبتدئ.
- `solution.py`: حل التمرين.
- `data/student_scores.csv`: بيانات تدريبية صغيرة.
- `notes.md`: خريطة الرياضيات التي تحتاجها في كل مرحلة.
- `requirements.txt`: المكتبة المطلوبة للتطبيقات.
- `lesson-guide.pdf`: كتيب PDF مختصر للدرس.

## التثبيت

يفضل العمل داخل بيئة افتراضية:

```bash
python -m venv .venv
```

### Windows

```bash
.venv\Scripts\activate
```

### macOS / Linux

```bash
source .venv/bin/activate
```

ثم ثبت المتطلبات:

```bash
python -m pip install -r requirements.txt
```

## التشغيل

```bash
python lesson_code.py
```

## الفكرة التعليمية

يمكنك البدء في الذكاء الاصطناعي بدون رياضيات قوية، لكن تحتاج إلى تقوية المفاهيم تدريجيًا:
1. المتوسطات والنسب.
2. الإحصاء الوصفي.
3. الاحتمالات الأساسية.
4. المتجهات والمصفوفات.
5. التفاضل لاحقًا عند دراسة التعلم العميق.

## ملاحظة

الأمثلة تعليمية فقط ولا تمثل نموذجًا حقيقيًا لاتخاذ قرارات عن الطلاب.
