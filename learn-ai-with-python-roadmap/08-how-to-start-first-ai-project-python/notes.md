# ملخص الدرس الثامن: أول مشروع ذكاء اصطناعي

## دورة المشروع

1. مشكلة واضحة.
2. بيانات مناسبة.
3. فحص البيانات وتنظيفها.
4. ميزات Features وهدف Target.
5. تقسيم Training / Testing.
6. تدريب نموذج.
7. تقييم النتيجة.
8. تحسين تدريجي.

## في هذا المشروع

- المشكلة: توقع درجة تقريبية.
- الميزة الأولى: ساعات الدراسة.
- الهدف: الدرجة.
- النموذج: LinearRegression.
- التقييم: Mean Absolute Error - MAE.

## معنى MAE

هو متوسط الفرق المطلق بين التوقعات والنتائج الحقيقية.
كلما كان أصغر ضمن نفس المشكلة والبيانات كان ذلك أفضل غالبًا.

## أفكار تطوير لاحقة

- أضف الحضور كميزة ثانية.
- أضف عدد الواجبات المنجزة.
- ارسم البيانات والتوقعات.
- جرب نموذجًا آخر.
- استخدم بيانات أكثر وأفضل.
