# ملخص الدرس السابع

## Python العادي

يركز على:
- المتغيرات والقوائم والدوال.
- الشروط والحلقات.
- الملفات والأتمتة.
- البرامج التي تعتمد على قواعد يكتبها المبرمج.

أمثلة:
- حاسبة عمر.
- منظم ملفات.
- مولد كلمات مرور.
- برنامج مصروفات.

## Python للذكاء الاصطناعي

يبدأ من نفس الأساس، ثم يضيف:
- CSV والجداول.
- Pandas وNumPy.
- الإحصاء والرسوم البيانية.
- ميزات Features وهدف Target.
- تدريب واختبار نموذج.
- توقع أو تصنيف.

أمثلة:
- تحليل مبيعات.
- توقع درجات.
- تصنيف رسائل.
- توصية منتجات.

## قاعدة المسار

لا يوجد Python خاص بالذكاء الاصطناعي.
اللغة واحدة، لكن المجال يحدد الأدوات والمشكلات والمشاريع التي ستعمل عليها.

## ترتيب مقترح

1. أساسيات Python.
2. قوائم ودوال وملفات.
3. CSV وPandas.
4. NumPy وMatplotlib.
5. scikit-learn ومشاريع تعلم آلة.
