# تعلم الذكاء الاصطناعي ببايثون من الصفر

هذا المجلد يحتوي على ملفات الدرس الأول من قسم الذكاء الاصطناعي في موقع **بايثون العرب**.

## رابط الدرس


https://www.arabpython.com/2026/06/learn-ai-with-python-from-zero-roadmap.html

## فكرة الدرس

الهدف من هذا الدرس هو توضيح الطريق الصحيح لتعلم الذكاء الاصطناعي ببايثون من الصفر، من أساسيات Python إلى فهم البيانات، ثم تحليل البيانات، ثم أول خطوة في Machine Learning.

## محتويات المجلد

```text
01-learn-ai-with-python-roadmap/
│
├── README.md
├── lesson-guide.pdf
├── lesson_code.py
├── practice.py
├── solution.py
├── notes.md
├── requirements.txt
└── data/
    └── students.csv
```

## الملفات

- `lesson-guide.pdf`: كتيب شرح مختصر للدرس.
- `lesson_code.py`: كود الدرس الأساسي.
- `practice.py`: تمرين تطبيقي للطالب.
- `solution.py`: حل التمرين.
- `notes.md`: ملاحظات ومصطلحات مهمة.
- `requirements.txt`: المكتبات المطلوبة للتشغيل.
- `data/students.csv`: ملف بيانات بسيط للتجربة.

## طريقة التشغيل

ثبت المكتبات أولًا:

```bash
pip install -r requirements.txt
```

ثم شغل كود الدرس:

```bash
python lesson_code.py
```

ولحل التمرين بنفسك افتح:

```bash
practice.py
```

ثم قارن إجابتك بالملف:

```bash
solution.py
```

## ماذا ستتعلم؟

- لماذا نبدأ بتعلم Python قبل الذكاء الاصطناعي.
- كيف تكون البيانات أساس مشاريع AI.
- كيف نقرأ ملف CSV بسيط باستخدام Pandas.
- كيف نحسب متوسطات بسيطة من البيانات.
- كيف نفكر في أول مشروع ذكاء اصطناعي صغير.

## ملاحظة

هذه الملفات تعليمية ومبسطة للمبتدئين، وليست مشروع ذكاء اصطناعي متقدم. الهدف هو بناء أول خطوة صحيحة في الطريق.
