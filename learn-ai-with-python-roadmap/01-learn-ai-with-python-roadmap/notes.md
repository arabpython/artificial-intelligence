# ملاحظات الدرس 01

## الفكرة الأساسية

الذكاء الاصطناعي لا يبدأ من النموذج مباشرة، بل يبدأ من فهم البيانات.

## مصطلحات مهمة

- **AI**: الذكاء الاصطناعي بشكل عام.
- **Machine Learning**: تعلم الآلة، وهو فرع من الذكاء الاصطناعي.
- **Dataset**: مجموعة بيانات يستخدمها النموذج للتعلم أو التحليل.
- **Model**: النموذج الذي يتعلم من البيانات أو يستخدمها للتوقع.
- **Training**: تدريب النموذج على البيانات.
- **Prediction**: التوقع أو النتيجة التي ينتجها النموذج.

## نصائح للمبتدئ

1. لا تبدأ بمشروع ضخم.
2. لا تحفظ الأكواد بدون فهم البيانات.
3. تعلم Python أولًا، ثم Pandas، ثم Machine Learning.
4. اجعل كل درس مرتبطًا بتمرين صغير.
5. ارفع ملفاتك على GitHub لتتعلم التوثيق والتنظيم.

## أخطاء شائعة

- القفز إلى Deep Learning قبل فهم Machine Learning.
- تجاهل تنظيف البيانات.
- الاعتماد على النسخ واللصق بدون تجربة.
- عدم فهم معنى الأعمدة والصفوف في البيانات.
