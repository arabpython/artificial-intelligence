# الفرق بين AI وMachine Learning وDeep Learning

حزمة تطبيقية مرافقة لدرس **الفرق بين الذكاء الاصطناعي وتعلم الآلة والتعلم العميق** من مسار **مفاهيم الذكاء الاصطناعي** في بايثون العرب.

## رابط الدرس

https://www.arabpython.com/2026/06/difference-between-ai-machine-learning-deep-learning.html

## ماذا ستتعلم؟

- الفرق بين النظام القائم على القواعد وMachine Learning.
- معنى Features وTarget في مشروع تعلم آلة صغير.
- تدريب نموذج Linear Regression بسيط باستخدام scikit-learn.
- لماذا لا نحتاج إلى Deep Learning في كل مشروع.

## محتويات الحزمة

- `rule_based_classifier.py`: مثال قرار قائم على قواعد مكتوبة يدويًا.
- `train_simple_ml.py`: تدريب نموذج تعلم آلة بسيط من بيانات CSV.
- `compare_models.py`: مقارنة منظمة بين AI وML وDL.
- `practice.py`: تمرين عملي.
- `solution.py`: حل التمرين.
- `data/study_scores.csv`: بيانات تعليمية مصطنعة.
- `notes.md`: ملخص الدرس.
- `lesson-guide.pdf`: كتيب شرح مختصر.
- `assets/`: صور المقال التعليمية.

## التشغيل

```bash
python -m pip install -r requirements.txt
python rule_based_classifier.py
python train_simple_ml.py
python compare_models.py
```

## تنبيه مهم

هذه الملفات والبيانات للتعلم فقط. لا تستخدم بيانات مصطنعة أو نموذجًا صغيرًا لاتخاذ قرارات حقيقية عن أشخاص.
